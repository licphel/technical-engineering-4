# Technical Engineering 4 (TEN4) for Minecraft 1.21 #

Technical Engineering 4 is an independent Minecraft tech mod built on NeoForge, featuring ore processing, energy
systems, and factory automation. The mod focuses on configurable, upgradeable machines with a flexible side-panel
configuration system.

Technical Engineering 4 is an official upgraded version of Technical Engineering 3, with its code completely rewritten.
